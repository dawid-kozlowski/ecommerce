# E-commerce Demo

A full-stack e-commerce application built with React (TypeScript) and Node.js (Express + Prisma).

## Project Structure

- **`/client`** - React frontend with TailwindCSS
- **`/server`** - Node.js REST API with Prisma ORM

## Quick Start

1. **Server**: `cd server && npm install && npm run dev`
2. **Client**: `cd client && npm install && npm run dev`

See individual README files in each directory for detailed documentation.
